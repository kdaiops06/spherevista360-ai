## What is SphereVista360?

**SphereVista360** is an open-source financial intelligence platform that combines live market data, AI-generated content, and programmatic SEO at scale. It is live at **[spherevista360.com](https://spherevista360.com)** — 6,200+ pages indexed by Google, serving real-time data from free APIs, and generating content daily via AI agents and GitHub Actions.

### Goal

Build a self-sustaining, revenue-generating financial content and tools platform that:

1. **Delivers real-time data** — Live market prices, currency rates, and financial news without requiring paid API keys
2. **Scales content through AI** — 6 AI agents produce daily articles on markets, currencies, and economics
3. **Ranks organically** — 6,200+ programmatic SEO pages capturing high-volume finance keywords
4. **Generates passive revenue** — Built-in monetization (ads, affiliates, newsletter, API access)
5. **Runs without maintenance** — GitHub Actions + Vercel auto-deploy keep everything up to date

### What's Live Today

| Feature | Status | Details |
|---|---|---|
| **Live market data** | ✅ Live | S&P 500, Dow, Nasdaq, Gold, Bitcoin, Treasuries via Yahoo Finance (free) |
| **Live currency rates** | ✅ Live | 150+ currencies via European Central Bank / frankfurter.app (free) |
| **Live financial news** | ✅ Live | Real-time headlines via Google News RSS (free) |
| **Currency converter API** | ✅ Live | 3-tier fallback: free ECB → paid ExchangeRate → static rates |
| **11 financial tools** | ✅ Live | Calculators, trackers, analyzers — all interactive |
| **6,200+ SEO pages** | ✅ Live | Currency pairs, inflation, investments, comparisons |

## AI Agents

Six specialized AI agents power content generation and analysis:

| Agent | What It Does | Trigger |
|---|---|---|
| `news-agent` | Fetches financial news → summarizes → generates articles | Daily cron |
| `currency-agent` | Analyzes exchange rates → produces forex commentary | Daily cron |
| `macro-agent` | Processes GDP, CPI, employment data → economic analysis | Daily cron |
| `prediction-agent` | Combines data sources → market outlook predictions | Daily cron |
| `seo-agent` | Generates metadata, schema markup, internal links | On article creation |
| `content-agent` | Produces comparison articles, inflation reports, investment guides | On demand |

### Agent Usage

```typescript
import { newsAgent, currencyAgent, contentAgent } from "./agents";

// Daily pipeline (automated via GitHub Actions)
const news = await newsAgent.fetchLatestNews();
const article = await newsAgent.generateNewsArticle(news);

// On-demand content
const comparison = await contentAgent.generateComparisonArticle("gold", "bitcoin");
const inflationReport = await contentAgent.generateInflationArticle("India", "2026");
```

## Automation Pipeline

Daily GitHub Actions workflow generates and publishes content automatically:

```
6:00 AM UTC (daily)
       │
       ├─ 1. Fetch news from News API + Reddit
       ├─ 2. Generate 3 articles (finance, currency, macro) via AI
       ├─ 3. Save as MDX in content/ directory
       ├─ 4. Git commit + push to main
       └─ 5. Vercel auto-deploys new content
```

**Manual trigger:**
```bash
gh workflow run auto-publish.yml -f article_count=5
```

**Run locally:**
```bash
npm run auto-publish   # Full pipeline: fetch → generate → publish
npm run fetch-news     # Just fetch news
npm run generate-article   # Just generate articles
```

## Environment Variables

```bash
cp .env.example .env.local
```

**Free data sources require no configuration** — Yahoo Finance, ECB, and Google News work out of the box.

| Variable | Required | Purpose |
|---|---|---|
| `ANTHROPIC_API_KEY` | Optional — for AI articles | Claude API for content generation |
| `NEWS_API_KEY` | Optional — free fallback exists | NewsAPI (higher quality news) |
| `ALPHA_VANTAGE_API_KEY` | Optional — free fallback exists | Stock/forex data (higher rate limits) |
| `EXCHANGERATE_API_KEY` | Optional — free fallback exists | Currency conversion (backup) |
| `FRED_API_KEY` | Optional — for economic data | GDP, CPI, employment indicators |
| `NEXT_PUBLIC_SUPABASE_URL` | Optional — for database | Supabase project URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Optional — for database | Supabase anonymous key |
| `NEXT_PUBLIC_ADSENSE_CLIENT_ID` | Optional — for ads | Google AdSense |

> **All variables are optional.** The platform serves live data from free APIs (Yahoo Finance, ECB, Google News RSS) with zero configuration. Paid keys unlock higher rate limits and additional data sources.

## SEO Strategy

See the full **[SEO Playbook →](SEO-PLAYBOOK.md)** for the technical breakdown of how 6,200+ pages are generated, indexed, and ranked.

Key strategies:
- **Programmatic page generation** — scripts create static params for every currency pair, country, and guide
- **JSON-LD schema markup** — FAQPage, WebApplication, and FinancialProduct schemas on every page
- **Internal linking mesh** — every page links to related pages (reverse pairs, same-category guides)
- **ISR (Incremental Static Regeneration)** — pages revalidate every 24 hours for fresh data
- **Automated content velocity** — daily AI articles compound organic keyword coverage
